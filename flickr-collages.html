Flickr Collages

Nikos Antonio Kourous Vázquez

Generative collages of famous paintings using machine learning object detection and images from Flickr. The project endlessly created collage reconstructions of paintings using found images on the web, creating humourous and at times very random compositions of famous scenes.

Shown at The Photographers Gallery // Flickr Foundation anniversary event 2025.
