Corrupted polygraph

Using my own code I reprogrammed a pen plotter to work like a polygraph by converting live heartbeat sensor data to x and y movement.
The first video shows it working without interference. The video below, however, shows the polygraph working while simultaneously processing data from a prisoner violence database.

Time and time again, cases arise of the police force using manipulative and illegal strategies to create false confessions and evidence within the legal system. The polygraph test is often seen as scientifically accurate and used as valid evidence. In reality, it's a scientific conduit for exploitation – its validity is extremely debated and there’s stronger consensus around it not being accurate.
My polygraph’s functions and accuracy is corrupted using data representing the shared anxiety that minority groups bear towards the police and prison system to highlight its role in perpetuating systemic imprisonment in the US.
